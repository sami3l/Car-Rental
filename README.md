<h1 align="center">Car Rental Desktop app</h1>

> This is a project for a car rental app. It is a project for the College Project.

# User Features

* Login
* Signup
* Search for cars
* Book cars


# Admin Features

* Login
* User Management
  * Add User
  * Update User
  * Delete User
  * Show all users
* Car Management
  * Add Cars
  * Update Cars
  * Delete Cars
  * Show all Cars
  
## Requirements
  ```sh
     Tkinter
     Mysqlconnector
  ```
